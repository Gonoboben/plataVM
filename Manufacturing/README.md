# Manufacturing

Раздел для производственных материалов ПДУ БНПА.

## Структура

```text
Manufacturing/
├─ BOM/
├─ Assembly/
└─ Release/
```

## Содержимое

1. финальный BOM с артикулами;
2. сборочная документация;
3. release notes;
4. комплект файлов для изготовления;
5. протоколы приёмки;
6. список известных ограничений ревизии.

Финальный BOM формируется только после выбора компонентов и испытаний.
