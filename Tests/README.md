# Tests

Раздел для стендовых испытаний, протоколов и журналов.

## Структура

```text
Tests/
├─ Bench/
├─ Power/
├─ Thermal/
├─ Fault_Injection/
└─ Logs/
```

## Правило

Испытания выполняются по `Docs/TEST_PLAN.md`.

Каждый протокол должен фиксировать:

1. дату;
2. ревизию платы;
3. ревизию прошивки;
4. схему стенда;
5. режим;
6. измерения;
7. результат;
8. дефекты;
9. вывод;
10. следующий шаг.
