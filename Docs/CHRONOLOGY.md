Проверка:

1. Репозиторий относится только к PlataVM.
2. BuoyCalc Windows исключён из области проекта.
3. Зафиксирован контекстный замок: `PlataVM / плата питания`.
4. Зафиксирована предварительная структура питания и список открытых вопросов.

Что специально не изменялось:

1. BuoyCalc Windows.
2. Solver BuoyCalc.
3. PDF/2D BuoyCalc.
4. Любые файлы другого репозитория.
5. Электрические схемы платы — пока не добавлены.
6. PCB — пока не добавлена.
7. BOM — пока не добавлен.
