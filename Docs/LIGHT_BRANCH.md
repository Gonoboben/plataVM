# LIGHT_POWER_BRANCH

## 1. Назначение

LIGHT_POWER_BRANCH питает шесть световых матриц Epistar XY-J45.

## 2. Legacy/current implementation

В существующей системе использовались TRACO POWER TEP 100-1216. В V1.5 они не являются целевой архитектурой новой платы.

## 3. Целевая архитектура

```text
PACK_BUS → LIGHT_POWER_BRANCH → onboard LED power / LED drivers → 6 × Epistar XY-J45
```

Предварительное разбиение:

```text
LED power module A → LED1...LED3
LED power module B → LED4...LED6
```

Шесть каналов остаются логически независимыми.

## 4. Требования к LED-каналу

1. Constant-current режим или контролируемое ограничение тока.
2. Расчётный максимум тока матрицы — 1,05 А.
3. PWM-диммирование.
4. Диапазон PWM: технически 5–100%, штатно 10–100%.
5. Измерение тока каждого канала.
6. Защита от КЗ.
7. Защита от обрыва.
8. Защита от перегрева.
9. Fault flag в MCU.
10. Разъём на каждый ИЗС.

## 5. Поведение по режимам

1. В RUN свет работает по командам MCU.
2. В DECK_BALANCE световая ветвь отключена.
3. В SAFE свет отключается одним из первых.
4. В KEEP_ALIVE и HARD_OFF световая ветвь обесточена.
5. В SINGLE_PACK_MODE общий лимит света должен ограничиваться энергетическим бюджетом одной батарейной линии.
