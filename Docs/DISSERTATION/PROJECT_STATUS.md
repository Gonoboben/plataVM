## 1. Рабочая тема

Предварительная тема кандидатской диссертации:

```text
Исследование и разработка энергетического обеспечения подводных буксируемых исследовательских комплексов
```

Специальность:

```text
1.6.17 — Океанология, технические науки
```

## 2. Инженерная основа работы

Работа строится вокруг энергетического обеспечения глубоководного буксируемого комплекса / БНПА «Видеомодуль».

Ключевая инженерная линия:

1. автономный аккумуляторный энергокомплекс;
2. распределение питания по нагрузкам;
3. питание осветителей, видеосистемы, датчиков и управляющей электроники;
4. резервное питание критических узлов;
5. телеметрия и управление с основного модуля;
6. испытания и внедрение в составе БНПА.

## 4. Зафиксированная архитектурная логика

Основное питание:

```text
BAT1 + BAT2 → параллельно → BAT_BUS_4S → силовые каналы / DC-DC → нагрузки
```

Резервное питание:

```text
EMG_4S2P → CRITICAL_INPUT → 5V_CRIT / 3V3_CRIT → критические узлы
```

Критическое ограничение:

```text
EMG_4S2P не питает ИЗС и BOOST 32 В
```

## 5. Уже зафиксированные проектные решения

1. Основные АКБ работают параллельно как главный источник питания.
2. Резервный источник является отдельной линией для критического питания.
3. Управление новыми функциями должно идти с компьютера через MCU.
4. Для связи с основным модулем используется RS-485.
5. UART / SWD используются как сервисные интерфейсы.
6. Power GND и Signal GND не объединяются произвольно.
7. Для проекта важна хронология: нельзя начинать заново, нужно модернизировать существующую архитектуру.
