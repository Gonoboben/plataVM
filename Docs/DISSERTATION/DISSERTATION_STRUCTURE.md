# Структура диссертации

Дата фиксации: 2026-06-30

Рабочая тема:

```text
Исследование и разработка энергетического обеспечения подводных буксируемых исследовательских комплексов
```

Специальность:

```text
1.6.17 — Океанология, технические науки
```

## 1. Введение

Во введении должны быть раскрыты:

1. актуальность темы исследования;
2. цель и задачи работы;
3. объект и предмет исследования;
4. методы исследования;
5. научная новизна;
6. практическая значимость;
7. положения, выносимые на защиту;
8. апробация результатов;
9. публикации по теме диссертации;
10. структура и объём диссертации.

### 1.1. Цель исследования

```text
Повышение эффективности проведения подводных исследований с использованием глубоководных буксируемых аппаратов за счёт разработки и внедрения системы энергетического обеспечения, обеспечивающей надёжное электропитание бортовых систем наблюдения при работе в условиях больших глубин и высоких энергозатрат.
```

### 1.2. Задачи исследования

1. Проанализировать современное состояние и опыт применения глубоководных подводных буксируемых носителей и систем их электропитания.
2. Теоретически обосновать требования к системе энергообеспечения глубоководного буксируемого комплекса и разработать структурную схему такой системы.
3. Разработать экспериментальный образец аккумуляторного энергокомплекса и алгоритмы его функционирования.
4. Создать стенд для испытаний и провести лабораторную отработку.
5. Провести натурные испытания энергетической системы в составе БНПА «Видеомодуль».
6. Проанализировать результаты внедрения системы в практику.

## 2. Глава 1. Современные буксируемые подводные исследовательские комплексы и системы их электропитания

Разделы главы:

1.1. Классификация и назначение подводных буксируемых систем.

1.2. Анализ отечественного и зарубежного опыта применения.

1.3. Обзор существующих систем энергетического обеспечения.

1.4. Проблематика и ограничения существующих решений.

Назначение главы:

```text
Показать существующую проблему и вывести требования к новой системе энергообеспечения.
```

Ожидаемый результат главы:

```text
обоснованный перечень требований к системе питания глубоководного буксируемого комплекса
```

## 3. Глава 2. Теоретическое обоснование требований и структуры системы энергообеспечения глубоководного буксируемого комплекса

Разделы главы:

2.1. Формализация требований к системе энергоснабжения.

2.2. Разработка структурной схемы системы питания.

2.3. Математическое моделирование энергетического баланса.

2.4. Анализ и выбор элементной базы для аккумуляторного комплекса.

Назначение главы:

```text
Перейти от требований к архитектуре системы питания и расчётной модели энергетического баланса.
```

Ожидаемый результат главы:

```text
структурная схема энергосистемы + модель энергетического баланса + обоснование элементной базы
```

## 4. Глава 3. Разработка экспериментального аккумуляторного комплекса и методика его испытаний

Разделы главы:

3.1. Конструктивная реализация аккумуляторного блока.

3.2. Стендовые и климатические испытания.

3.3. Анализ результатов и корректировка конструкции.

Назначение главы:

```text
Показать инженерную реализацию, испытания и корректировку конструкции.
```

Ожидаемый результат главы:

```text
экспериментальный образец + методика испытаний + подтверждение работоспособности
```

## 5. Глава 4. Натурные испытания и внедрение энергетической системы в составе БНПА «Видеомодуль»

Разделы главы:

4.1. Организация и методика натурных испытаний.

4.2. Результаты экспедиционных исследований.

4.3. Оценка эффективности и эксплуатационные показатели.

4.4. Внедрение и перспективы применения.

Назначение главы:

```text
Подтвердить работоспособность и эффективность системы в составе реального БНПА.
```

Ожидаемый результат главы:

```text
результаты испытаний + эксплуатационная оценка + подтверждение внедрения
```

## 6. Заключение

В заключении должны быть зафиксированы:

1. основные результаты и выводы по работе;
2. достижение цели;
3. решение поставленных задач;
4. научный вклад;
5. практическая значимость;
6. ограничения и направления дальнейших исследований.

## 7. Приложения

Предварительный состав приложений:

1. структурная схема системы питания;
2. электрические схемы;
3. BOM;
4. расчёт энергетического баланса;
5. методика испытаний;
6. протоколы испытаний;
7. фотографии конструкции;
8. акты внедрения;
9. список сокращений и условных обозначений.

## 8. Правило дальнейшей работы

Каждый новый инженерный документ должен быть связан с конкретным местом в диссертации:

```text
документ → глава → задача исследования → инженерный результат → подтверждение
```
