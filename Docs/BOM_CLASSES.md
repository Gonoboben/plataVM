# Классы компонентов и BOM-логика ПДУ БНПА / PlataVM

## 1. Компоненты внутри корпуса каждой основной АКБ

1. Аккумуляторная сборка LiFePO4 4s24p.
2. Существующая автономная BMS.
3. Батарейный предохранитель.
4. Однополюсный контактор K_BATx.
5. Пассивный элемент подавления выброса катушки.
6. Внутренняя силовая проводка.
7. Блочная часть СН-176А-12.

В корпусе АКБ не предусматриваются:

1. отдельный MCU;
2. RS-485/CAN/UART-трансивер;
3. интерфейсная плата;
4. дополнительный ADC;
5. цифровой канал BMS;
6. отдельный DC/DC для электроники.

## 2. Межкорпусная батарейная линия

1. СН-176А-12.
2. Кабельная сборка длиной 1 м.
3. Пять параллельных жил/контактов BAT+.
4. Пять параллельных жил/контактов BAT−.
5. Одна или две линии управления K_BATx.

## 3. Компоненты в корпусе электроники

1. Battery Front-End #1/#2.
2. TVS и EMI-защита батарейных входов.
3. Измерение BAT1_V/BAT1_I и BAT2_V/BAT2_I.
4. Драйверы K_BAT1/K_BAT2.
5. MAIN_SW1/MAIN_SW2.
6. BALANCE_SW1/BALANCE_SW2.
7. R_BAL1/R_BAL2.
8. PACK_BUS discharge circuit.
9. Разъёмы POWER_12V_BUS CH1...CH14.
10. High-side switches/eFuse для CH1...CH14.
11. DC/DC 12→5 В для 5V_SYS_BUS.
12. Разъёмы 5V_OUT1...5V_OUT10.
13. Защита и токовая диагностика каждого 5V_OUT.
14. Onboard LED drivers для 6 × Epistar XY-J45.
15. MCU.
16. Isolated RS-485 transceiver.
17. Optional CAN transceiver footprint.
18. Watchdog/supervisor.
19. 5V_CRIT и 3V3_CRIT.
20. Test points и силовые межплатные соединения.

Центральные K_MAIN и глобальный PRECHARGE после PACK_BUS в BOM не включаются.

## 4. Статусы BOM

```text
candidate / selected / tested / approved / replaced / obsolete
```

## 5. Правила выбора K_BAT1/K_BAT2

1. Одинаковая модель для обеих АКБ.
2. DC breaking capacity должна соответствовать реальному току линии.
3. Катушка должна допускать удалённое управление через СН-176А-12.
4. Тепловыделение катушки должно быть допустимо в герметичном корпусе.
5. Тип контактора должен однозначно задавать функцию контактов 11–12.
6. HARD_OFF не должен зависеть от цифровой связи.
7. TRACO POWER относится только к legacy/current implementation.
