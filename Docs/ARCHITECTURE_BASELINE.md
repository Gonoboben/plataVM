# Архитектурная база ПДУ БНПА / PlataVM

Статус:

```text
V1.5 integrated TO-BE baseline
```

## 1. Назначение

Файл фиксирует базовую архитектуру питания, управления, резервирования и распределения выходных каналов.

## 2. Исходные данные

| Параметр | Значение | Статус |
|---|---:|---|
| АКБ_1 | LiFePO4 4s24p | принято |
| АКБ_2 | LiFePO4 4s24p | принято |
| Номинальное напряжение | 12,8 В | принято |
| Суммарная ёмкость основных АКБ | 288 А·ч | принято |
| EMG / АКБ_RES | LiFePO4 4s2p | принято |
| Ёмкость EMG | 12 А·ч | рабочее допущение |
| ИЗС | 6 × Epistar XY-J45 | принято |
| Мощность одного ИЗС | 30 Вт | принято |
| Основной интерфейс | isolated RS-485 | принято |
| CAN/CAN-FD | optional future | принято |
| POWER_12V_BUS | 14 каналов | принято |
| 5V_SYS_BUS | отдельная внешняя 5 В шина | принято |

## 3. Базовые шины

| Шина | Источник | Назначение |
|---|---|---|
| PACK_BUS | АКБ_1/АКБ_2 через Battery Front-End | объединение основных АКБ |
| MAIN_INPUT_BUS | PACK_BUS через BATTERY_DISCONNECT/PRECHARGE | главная входная шина платы |
| POWER_12V_BUS | MAIN_INPUT_BUS | 14 внешних 12 В каналов |
| 5V_SYS_BUS | MAIN_INPUT_BUS через DC/DC 12→5 В | внешние 5 В выходы |
| LIGHT_POWER_BRANCH | MAIN_INPUT_BUS | питание световых матриц |
| RESERVE_BRANCH | MAIN_INPUT_BUS + EMG | keep-alive и критическое питание |
| 5V_CRIT | RESERVE_BRANCH / critical power path | MCU, связь, датчики, аварийная логика |
| 3V3_CRIT | 5V_CRIT | цифровая логика |

## 4. Запрещённые связи

1. EMG не питает POWER_12V_BUS.
2. EMG не питает 5V_SYS_BUS.
3. EMG не питает LIGHT_POWER_BRANCH.
4. POWER_12V_BUS не питает LIGHT_POWER_BRANCH.
5. 5V_SYS_BUS не должна быть единственной шиной питания MCU.
6. TRACO не является целевой архитектурой новой платы.
7. CAN/CAN-FD не становится baseline без отдельного ADR.
