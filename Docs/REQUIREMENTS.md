# Требования к ПДУ БНПА / PlataVM

Дата фиксации: 2026-06-30

Статус:

```text
requirements baseline for V1.4 chat-consolidated architecture
```

## 1. Назначение

Документ фиксирует требования к плате питания БНПА.

Требования используются для:

1. схемотехнического freeze;
2. выбора компонентов;
3. PCB-разводки;
4. разработки прошивки;
5. подготовки тест-плана;
6. проверки готовности изделия.

## 2. Верхнеуровневое требование

Плата должна обеспечивать безопасное распределение энергии от двух основных аккумуляторных сборок и отдельного аварийного источника на силовые, вспомогательные и критические нагрузки БНПА.

## 12. Критерий готовности требований

Требования считаются пригодными для перехода к PCB, когда:

1. закрыты требования со статусом `открыто`, влияющие на схему;
2. выполнен расчёт токов и тепла;
3. выбраны компоненты для всех силовых и критических узлов;
4. сформирован тест-план, связанный с каждым требованием;
5. обновлены `ARCHITECTURE_BASELINE.md` и `OPEN_QUESTIONS.md`.
