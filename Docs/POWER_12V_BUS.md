# POWER_12V_BUS

## 1. Назначение

POWER_12V_BUS — внешняя 12 В шина бортовой электроники.

## 2. Источник питания

```text
MAIN_INPUT_BUS → POWER_12V_BUS → CH1...CH14
```

## 3. Канальная структура

| Каналы | Тип | Назначение |
|---|---|---|
| CH1...CH11 | MCU-controlled | назначение TBD |
| CH12...CH14 | Always-On | назначение TBD |

## 4. Требования к каждому каналу

1. Разъём на выходе платы.
2. Индивидуальный ключ или защищённый high-side switch.
3. Защита от КЗ.
4. Защита от перегрузки.
5. Статус FAULT.
6. Желательно измерение тока.
7. Управление MCU для CH1...CH11.
8. Мониторинг Always-On для CH12...CH14.

## 5. Поведение режимов

| Режим | POWER_12V_BUS |
|---|---|
| OFF | выключена |
| BOOT | выключена до разрешения |
| RUN | включается по каналам |
| SAFE | некритические каналы отключаются |
| KEEP-ALIVE | выключена |
| HARD OFF | физически обесточена |
