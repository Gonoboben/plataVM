# Состояния, аварийная логика и реакции ПДУ БНПА / PlataVM

Статус:

```text
state-machine baseline for V1.5 battery architecture clarification
```

## 1. Состояния

| Состояние | Назначение |
|---|---|
| OFF | основные батарейные ветви отключены |
| BOOT | запуск critical domain и MCU |
| SELF_TEST | самопроверка |
| DECK_BALANCE | палубное межблочное выравнивание АКБ_1/АКБ_2 и обслуживание EMG |
| RUN | штатная параллельная работа обеих АКБ |
| SAFE | ограничение нагрузки перед штатным отключением |
| KEEP_ALIVE | питание только критического домена от EMG |
| HARD_OFF | физическое отключение K_BAT1 и K_BAT2 |
| FAULT_LATCH | аварийная защёлка |

Отдельного состояния `BATTERY_MISMATCH` нет.

## 2. RUN

В `RUN` обе исправные АКБ работают параллельно через симметричные Battery Front-End. Различие напряжений, токов и SoC используется для телеметрии и расчёта состояния, но само по себе не вызывает отключения батареи.

## 3. DECK_BALANCE

В режиме `DECK_BALANCE`:

1. LIGHT_POWER_BRANCH отключена;
2. POWER_12V_BUS отключена либо переведена в сервисный минимум;
3. 5V_SYS_BUS отключена либо ограничена сервисными нагрузками;
4. MCU контролирует BAT1_V, BAT2_V, BAT1_I, BAT2_I, температуры и SoC;
5. разрешается ограниченный межблочный балансировочный ток;
6. операция прекращается при превышении тока, температуры или времени;
7. при необходимости выполняется заряд и диагностика EMG.

## 4. Штатный HARD_OFF

```text
SAFE
→ отключить тяжёлые нагрузки
→ сохранить журнал и SoC
→ K_BAT1 OFF
→ K_BAT2 OFF
→ проверить BAT1_I = 0 и BAT2_I = 0
→ проверить разряд PACK_BUS
→ завершить работу MCU от EMG
```

## 5. Аварийный HARD_OFF

При `EXT_KILL` или критической аварии:

1. электронные силовые тракты батарейных ветвей отключаются немедленно;
2. подаётся команда OFF на K_BAT1 и K_BAT2;
3. причина события сохраняется после отключения основных АКБ за счёт EMG;
4. контролируется разряд PACK_BUS.

## 6. Запрещено

1. Питать POWER_12V_BUS от EMG.
2. Питать 5V_SYS_BUS от EMG.
3. Питать LIGHT_POWER_BRANCH от EMG.
4. Игнорировать EXT_KILL.
5. Выполнять межблочное выравнивание без ограничения тока.
6. Выполнять штатное выключение без проверки токов батарейных ветвей и PACK_BUS.

## 7. Аварийные реакции

| Авария | Реакция |
|---|---|
| EXT_KILL | немедленный аварийный HARD_OFF |
| Overcurrent BATx | отключить электронный тракт и соответствующий K_BATx |
| Contactor feedback fault | отключить электронный тракт, зафиксировать отказ |
| Balance overcurrent | прекратить DECK_BALANCE |
| Balance overtemperature | прекратить DECK_BALANCE |
| Balance timeout | прекратить DECK_BALANCE, сохранить событие |
| Overcurrent CHx | отключить канал, записать fault |
| 5V_SYS fault | отключить 5V_SYS, сохранить critical domain |
| LED fault | отключить аварийный LED-канал |
| RS-485 timeout | перейти в безопасную политику |
| EMG undervoltage | ограничить keep-alive, записать fault |
