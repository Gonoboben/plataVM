# PlataVM

Аппаратный проект платы питания / энергетического модуля БНПА.

Дата инициализации репозитория: 2026-06-30

## Контекстный замок

Текущий активный проект:

```text
PlataVM / плата питания
```

Этот репозиторий не является частью BuoyCalc Windows.

```text
PlataVM ≠ BuoyCalc Windows
```

## Назначение репозитория

Репозиторий предназначен для ведения инженерной разработки платы питания и связанных документов:

1. архитектура питания;
2. структурные схемы;
3. электрические схемы;
4. выбор элементной базы;
5. расчёты токов, мощностей, тепла и защит;
6. BOM;
7. тест-планы;
8. протоколы испытаний;
9. хронология изменений;
10. ADR — журнал архитектурных решений.

## Что не входит в проект

В этот репозиторий не входит:

1. BuoyCalc Windows;
2. расчёт морской постановки;
3. PDF/2D-отчёты BuoyCalc;
4. solver BuoyCalc;
5. пользовательское Windows-приложение BuoyCalc.

## Базовые документы

1. `Docs/PROJECT_CONTROL.md` — правила ведения проекта.
2. `Docs/ARCHITECTURE_BASELINE.md` — архитектурная база PlataVM.
3. `Docs/CHRONOLOGY.md` — хронология проекта.
4. `Docs/adr/` — архитектурные решения.
5. `.github/PULL_REQUEST_TEMPLATE.md` — шаблон PR.

## Правило работы

После первичной инициализации любые изменения выполняются через отдельные ветки и pull request.

Перед изменением схемы питания, интерфейсов, земли, защит, аккумуляторной архитектуры или аварийного питания необходимо обновить архитектурную базу или добавить ADR.
