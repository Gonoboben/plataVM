# PCB

Раздел для печатной платы ПДУ БНПА.

PCB не должна разрабатываться до закрытия схемотехнического freeze.

## Перед началом PCB обязательно

1. Закрыть вопросы из `Docs/OPEN_QUESTIONS.md`, влияющие на схему.
2. Утвердить электрическую схему.
3. Выполнить расчёты токов и тепла.
4. Свериться с `Docs/PCB_RULES.md`.
5. Обновить `Docs/CHRONOLOGY.md`.

## Базовые ограничения

1. EMG не питает ИЗС.
2. EMG не питает BOOST 32 В.
3. BAT1/BAT2 включаются через контроль и precharge.
4. Power GND и Signal GND не объединяются произвольно.
